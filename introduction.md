# This is the introduction file for System Design Notes

## Challenges that we solve in high-level system design
- Optimise the process and increase the throughput with in the same resources.(**Vertical scaling**)
- Preparing before hand during non-peak hours to handle the peak hours. (**Preprocessing and Cron jobs**)
- Hire More resources to handle the peak hours. (**Horizontal scaling**)
- Microservices architecture to handle the peak hours. (**Microservices**)
- Distributed systems to handle the peak hours. (**Distributed Systems**)
- Load balancing to distribute the load evenly across the resources. (**Load Balancing**)
- Caching to reduce the load on the resources. (**Caching**)
- Extensibility to handle the peak hours. (**Extensibility**)
